🎓 Student Management System (BTL--CD2)

- Ứng dụng quản lý sinh viên được xây dựng trên nền tảng Flutter, giúp quản lý thông tin sinh viên một cách hiệu quả, trực quan và dễ sử dụng. Đây là đồ án môn học (BTL) tập trung vào việc áp dụng kiến thức về lập trình di động và quản lý dữ liệu.

🚀 Tính năng chính

- Quản lý danh sách lớp: Hiển thị danh sách lớp học.

- Quản lý sinh viên: Hiển thị thông tin sinh viên.

- Quản lý ngành học: Hiển thị danh sách ngành học.

- Quản lý môn học: Hiển thị danh sách môn học.

- Thêm, sửa, xóa sinh viên mới: Nhập thông tin chi tiết (Tên, MSSV, Lớp, Điểm...).

- Tìm kiếm: Hỗ trợ tìm kiếm sinh viên nhanh theo tên hoặc mã số.

🛠 Công nghệ sử dụng

- Framework: Flutter

- Ngôn ngữ: Dart

- Cơ sở dữ liệu: SQLite 

- Kiến trúc: MVC

⚙️ Cài đặt và Chạy ứng dụng

Để chạy project này trên máy cục bộ, hãy làm theo các bước sau:

Clone repository:

Bash
git clone https://github.com/cuongcuong1234/BTL--CD2.git

Di chuyển vào thư mục dự án:

Bash
cd BTL--CD2/my_app

Cài đặt các thư viện (dependencies):

Bash
flutter pub get

Chạy ứng dụng:

Bash
flutter run
